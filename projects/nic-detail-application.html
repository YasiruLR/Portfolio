NIC Detail Application

A React application that extracts and displays personal details from Sri Lankan National Identity Card numbers using the lanka-nic library. Features responsive UI with custom reusable components.

React 19 Vite JavaScript ES6+ CSS3

Project Overview

The NIC Detail Application is a specialized React-based tool designed to parse and extract information from Sri Lankan National Identity Card (NIC) numbers. This application demonstrates the power of modern React development combined with specialized library integration for real-world utility.

Built with React 19 and Vite for optimal development experience, the application features a clean, responsive interface with custom reusable components, form validation, and real-time data parsing capabilities for extracting birth date and gender information from NIC numbers.

Key Features

NIC Processing

  • Parse and validate Sri Lankan NIC numbers
  • Extract birth date information
  • Determine gender from NIC
  • Real-time data parsing and validation

User Interface

  • Modern and responsive user interface

  • Interactive feedback and animations
