SmartBiz

AI-Powered Business Management Suite

A comprehensive ERP-lite system for SMEs with AI-powered insights using OpenAI integration, featuring React web app, React Native mobile app, and microservices-based Spring Boot backend.

React React Native Spring Boot MySQL OpenAI API AWS EC2

Project Overview

SmartBiz represents the next generation of business management solutions, combining traditional ERP functionality with cutting-edge AI capabilities. This comprehensive system is designed specifically for Small and Medium Enterprises (SMEs) who need powerful business tools without the complexity of enterprise-grade systems.

The project utilizes a modern microservices architecture with Spring Boot backend, complemented by both web and mobile frontend applications. The integration of OpenAI's API enables natural language reporting and intelligent business insights, setting it apart from traditional business management systems.
